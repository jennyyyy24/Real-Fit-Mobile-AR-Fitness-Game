# RealFit - AI-Powered Exercise Gaming App

RealFit is an innovative Flutter mobile application that combines fitness with gaming using real-time pose detection. Transform your workout routine into an engaging gaming experience while tracking your fitness progress and burning calories.

## 🎯 What is RealFit?

RealFit gamifies exercise by using your device's camera and Google ML Kit's pose detection to track your movements in real-time. The app turns traditional exercises into interactive games, making fitness fun, engaging, and motivating.

## ✨ Key Features

### 🎮 Interactive Exercise Games
- **Dinosaur Squat (Flappy Bird Style)**: Jump through obstacles by performing squats
- **Tuck Jump Challenge**: Navigate through levels with explosive tuck jumps

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (>=3.1.3)
- Android Studio / Xcode for mobile development
- Device with camera capability
- Good lighting for optimal pose detection

### Installation
1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Configure Supabase credentials in your environment
4. Run `flutter run` to start the application

### First Time Setup
1. Create an account or sign in with Google
2. Complete the onboarding process to set your fitness goals
3. Allow camera permissions for pose detection
4. Choose your first exercise game and start your fitness journey!
